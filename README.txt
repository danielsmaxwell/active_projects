See also Jeff Leek eCOTS presentation.

Facets of Data Storytelling
        Impart statistical concepts in data stories.
        Provide students with the tools to tell data stories.
        Help students become intelligent consumers of data stories.

Topics (D. Valle) -----------------------------------------------------
key computer programming concepts
- loops
- flow control
- customized functions
- indexing
- boolean logic
- vectorized operations

key R concepts
- read and write files, set working directory
- visualization: creating base R graphs + ggplot, saving graphs
- data types (character, numeric, logic)
- types of objects (dataframes, lists, matrices, vectors, arrays)
- manipulation of strings (grep, strsplit, paste)
- math operators and logical operators(e.g., &, |, %in%) 

other useful concepts and ideas:
- optimization
- sample(), random number generator, set.seed()
